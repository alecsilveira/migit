Ejemplos solución de E.D de orden superior con Máxima y Matlab

Los comandos ode2(), desolve() y contrib_ode() en Máxima y dsolve() en Matlab, nos ayudan a resolver ecuaciones diferenciales ordinarias de orden superior.

Ejemplo 1:

Resolver la E.D:

Solución con Máxima:

ed:'diff(y,x,2)+2*'diff(y,x)+2*y=%e^-x+x^2+sin(x)$
y1:ode2(ed,y,x);
y=(%e^(-x)*(2*%e^x*sin(x)-4*%e^x*cos(x)+(5*x^2-10*x+5)*%e^x+10))/10+%e^(-x)*(%k1*sin(x)+%k2*cos(x))

Observe que la 2da derivada se escribe como: ‘diff(y,x,2) y la 1era como:‘diff(y,x). Para ejecutar el comando es necesario presionar la combinación de teclas SHIFT+ENTER. Adicionalmente podemos mencionar que si una entrada se finaliza con el signo “$” entonces no se desplegarán resultados de esta.

La E.D También se puede resolver usando el comando desolve():

ed:'diff(y(x),x,2)+2*'diff(y(x),x)+2*y(x)=%e^-x+x^2+sin(x)$
y1=desolve(ed,y(x));
y(x)=%e^(-x)*((sin(x)*((2*(5*(at('diff(y(x),x,1),x=0))+10*y(0)-2))/5-(10*y(0)-11)/5))/2+((10*y(0)-11)*cos(x))/10)+sin(x)/5-(2*cos(x))/5+%e^(-x)+x^2/2-x+1/2

Observe que cuando se usa desolve(), la función a encontrar en la E.D se escribe como y(x), mientras que cuando se usa ode2() se escribe solo como y.

Solución con Matlab:

>> dsolve('D2y+2*Dy+2*y=exp(-x)+x^2+sin(x)','x')
 ans =
 (sin(x)*(sin(x) - (exp(x)*cos(x))/2 + (exp(x)*sin(x))/2 - (cos(2*x)*exp(x))/5 + (sin(2*x)*exp(x))/10 - x*exp(x)*sin(x) + (x^2*exp(x)*cos(x))/2 + (x^2*exp(x)*sin(x))/2))/exp(x) + (cos(x)*(cos(x) - exp(x)/2 + (exp(x)*cos(x))/2 + (exp(x)*sin(x))/2 + (cos(2*x)*exp(x))/10 + (sin(2*x)*exp(x))/5 - x*exp(x)*cos(x) + (x^2*exp(x)*cos(x))/2 - (x^2*exp(x)*sin(x))/2))/exp(x) + (C2*cos(x))/exp(x) + (C3*sin(x))/exp(x)

Como puede observarse, en Matlab la 2da derivada se denota como: D2y y la 1era como:Dy.

Ejemplo 2:

Resolver la E.D anterior con las condiciones iniciales: y .  Realice una gráfica de la solución hallada en el rango -6<x<6.

Solución con Máxima:

ed2:'diff(y,x,2)+2*'diff(y,x)+2*y=%e^-x+x^2+sin(x)$
y2:ode2(ed2,y,x)$
ic2(y2,x=0,y=0, 'diff(y,x)=0);
y=(%e^(-x)*(2*%e^x*sin(x)-4*%e^x*cos(x)+(5*x^2-10*x+5)*%e^x+10))/10+%e^(-x)*((7*sin(x))/10-(11*cos(x))/10)

Observe que para el comando ode2(), las condiciones iniciales se introducen con ic2().

Solución usando desolve():

ed2:'diff(y(x),x,2)+2*'diff(y(x),x)+2*y(x)=%e^-x+x^2+sin(x)$
atvalue(y(x),x=0,0)$
atvalue('diff(y(x),x),x=0,0)$
y2:desolve(ed2,y(x));
y(x)=sin(x)/5+%e^(-x)*((7*sin(x))/10-(11*cos(x))/10)-(2*cos(x))/5+%e^(-x)+x^2/2-x+1/2

En este caso, para el comando desolve(), las condiciones iniciales se introducen con atvalue().

Ambas soluciones son las mismas.  Para verificar, expandimos los términos en la solución obtenida con ode2(), usando el comando expand():

expand(%)

y=(7*%e^(-x)*sin(x))/10+sin(x)/5-(11*%e^(-x)*cos(x))/10-(2*cos(x))/5+%e^(-x)+x^2/2-x+1/2

Para hacer la gráfica, primero definimos la variable a graficar y luego usamos el comando plot2d():

y:(7*%e^(-x)*sin(x))/10+sin(x)/5-(11*%e^(-x)*cos(x))/10-(2*cos(x))/5+%e^(-x)+x^2/2-x+1/2$
plot2d(y,[x,-6,6]);

La gráfica obtenida es:

Solución con Matlab:

>>y= dsolve('D2y+2*Dy+2*y=exp(-x)+x^2+sin(x)','y(0)=0','Dy(0)=0','x')
ans = 
(7*sin(x))/(10*exp(x)) - (11*cos(x))/(10*exp(x)) + (sin(x)*(sin(x) - (exp(x)*cos(x))/2 + (exp(x)*sin(x))/2 - (cos(2*x)*exp(x))/5 + (sin(2*x)*exp(x))/10 - x*exp(x)*sin(x) + (x^2*exp(x)*cos(x))/2 + (x^2*exp(x)*sin(x))/2))/exp(x) + (cos(x)*(cos(x) - exp(x)/2 + (exp(x)*cos(x))/2 + (exp(x)*sin(x))/2 + (cos(2*x)*exp(x))/10 + (sin(2*x)*exp(x))/5 - x*exp(x)*cos(x) + (x^2*exp(x)*cos(x))/2 - (x^2*exp(x)*sin(x))/2))/exp(x)

Para simplificar el resultado, usamos el comando simplify():

>>simplify(y)
ans = 
1/exp(x) - x - (2*cos(x))/5 + sin(x)/5 - (11*cos(x))/(10*exp(x)) + (7*sin(x))/(10*exp(x)) + x^2/2 + 1/2

Para graficar directamente la solución, usamos el comando ezplot(y). La gráfica obtenida es:

Ejemplo 3:

Resolver la E.D:

Solución con Máxima:

ed3:x^2*'diff(y,x,2)+2*'diff(y,x)-y=0$
yc3:ode2(ed2,y,x);
false

El mensaje “false” nos indica que el comando ode2() no puede solucionar la E.D.  Intentamos con desolve():

ed3:x^2*'diff(y(x),x,2)+2*'diff(y(x),x)-y(x)=0$
yc3:desolve(ed2,y(x));
y(x)=ilt(-(g2195^2*('diff(laplace(y(x),x,g2195),g2195,2))+4*g2195*('diff(laplace(y(x),x,g2195),g2195,1))-2*y(0))/(2*g2195+1),g2195,x)

Ya que desolve() resuelve la E.D, mediante la transforma inversa de Laplace, en este caso la deja indicada y por tanto no hay solución explícita.

Para solucionar la E.D, podemos acudir al comando contrib_ode() que extiende las funcionalidades de ode2.  Primero debemos cargar el módulo que contiene dicho comando usando: load(contrib_ode), para luego usarlo:

ed3:x^2*'diff(y,x,2)+2*'diff(y,x)-y=0$
load(contrib_ode)$
yc3:contrib_ode(ed3,y,x);
[y=kummer_u((sqrt(5)+3)/2,sqrt(5)+1,-2/x)*%k2*x^(-sqrt(5)/2-1/2)*%e^(2/x)*abs(x)+kummer_m((sqrt(5)+3)/2,sqrt(5)+1,-2/x)*%k1*x^(-sqrt(5)/2-1/2)*%e^(2/x)*abs(x)]

El resultado son funciones “kummer“.

Solución con Matlab:

>> dsolve('x^2*D2y+2*Dy-y=0','x')
 ans =
 C8*x*exp(2/x)^(1/2)*whittakerM(1, -5^(1/2)/2, 2/x) + C9*x*exp(2/x)^(1/2)*whittakerW(1, -5^(1/2)/2, 2/x)

En este caso, la solución es expresada en función de las funciones de whittaker.
