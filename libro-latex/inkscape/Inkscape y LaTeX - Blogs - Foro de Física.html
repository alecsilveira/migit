Ya hace algún tiempo que empecé a usar Inkscape, a mi criterio es uno de los mejores programas libres para realizar dibujos vectoriales y con una calidad bastante buena, además es multiplataforma y bastante ligero en comparación de programas comerciales similares a este.

Algunas imágenes que se pueden crear usando este programa son por ejemplo



todos ellos hechos por mi en algún momento para algún post del foro.

Aunque siempre han habido cosas que he querido hacer con ese programa pero no las he logrado, como por ejemplo al hacer una flecha y que la cabeza de esta sea del color de la línea de la flecha y también poder escribir ecuaciones, pues en algunos diagramas resulta útil incluir alguna. Lo primero no he logrado aprender a hacerlo , pero lo segundo si, y a continuación procederé a explicarlo brevemente.

Existe un complemento llamado textex, el cual se lo puede descargar de acá para Windows y de acá para Linux, su forma de instalación es bastante sencilla, y explicaré para la versión 0.46 o posteriores de Inkscape.

Primero se necesita tener instalado , para que el complemento pueda funcionar, una vez instalado es necesario tener en cuenta lo siguiente:

En Linux, tener instalado Pstoedit o Pdf2svg, a mi me funcionó con el segundo, para instalarlo es muy sencillo, se lo puede hacer desde consola escribiendo

Código:
sudo apt-get install pstoedit
sudo apt-get install pdf2svg
y además descomprimir el archivo de textex en ~/.config/inkscape/extensions/.

En Windows, además de instalar Pstoedit es necesario tener Ghostscript, lo cual es como redundante decirlo debido a que si se suele usar seguramente ya lo tienen instalado en el ordenador.

Una vez finalizada la instalación abrir Inkscape y en la penúltima pestaña que tiene el nombre de "extensiones", se puede encontrar al complemento recientemente instalado, al darle click se abrirá una ventana como se muestra en la siguiente figura



en la parte superior se observa un espacio para colocar el preámbulo que se va a usar para escribir usando , para ello tienen que crear un archivo con extensión .inc y guardarlo en algún lugar de su ordenador, en mi caso el contenido de mi archivo preambulo.inc es

Código:
\usepackage{latexsym}
\usepackage{amssymb} 
\usepackage{amsmath}
\usepackage{amsthm}
\usepackage{cancel}
\newcommand{\dd}{\mathrm{d}}
\newcommand{\dst}{\displaystyle}
finalmente solo queda indicar que los criterios para escribir ecuaciones son los mismos que cuando se escriben en , por decirlo de un modo simple, las ecuaciones se escriben entre signos de dolar ($ecuación$).

Espero que este artículo les sea de utilidad, y si consideran que hay algo más que comentar acerca de esto, adelante!!
